# 概述

## 什么是 Preezid?

`Preezid` 是一个全新的轻量级前端框架, 旨在将前端从 `xml` 中完全脱离出来, 使用纯粹的函数式进行编程.

下面是一个简单的示例:

```lts
define {
  const count = status<number>(0)
  const increment = () => count.value++
}

load {
  div {
    button.@click(increment) { "Count is:" count.value }
  }
}
```

上述示例展示了 `Preezid` 的两个特性:

- **函数式**: `Preezid` 使用类似 `JavaScript/TypeScript` 的语法, 完全脱离了 `xml` 冗长的语法.
- **响应式**: 通过 `status` 函数创建响应式状态, 当状态变化时, 界面会自动更新.
- **轻量**: 拓展 `ts` 语法更易于书写.

## `Label JavaScript/TypeScript` 是什么?

`Label JavaScript/TypeScript` 是 `Preezid` 的核心语言, 它是 `JavaScript/TypeScript` 的一个超集, 添加了一些特性以支持函数式编程和响应式编程.

`Label JavaScript/TypeScript` 文件类型为 `.ljs/.lts`.

下面以 `Label TypeScript` 为例, 展示了 `Label JavaScript/TypeScript` 的一些特性:

```lts
load {
  div.class("container-1") {
    for (const item of items) {
      span { item.name }
    }
  }
  
  div.class("container-1") {
    arr.forEach(item => {
      span { item.name }
    })
  }
  
  button { "Click me !" }
}
```

上述示例展示了 `Label JavaScript/TypeScript` 的一些特性:

- **标签化**: `Label JavaScript/TypeScript` 使用预设的标签来表示界面元素, 每个标签将对应一个元素.
- **完全`js/ts`支持**: `Label JavaScript/TypeScript` 完全支持 `JavaScript/TypeScript` 的语法和特性, 可以使用所有的 `js.ts` 功能, 即使在标签中.

## 单文件组件

与 `vue` 和 `svelet` 类似, `Preezid` 组件也是单文件组件, 每个组件为一个 `.ljs/.lts` 文件, 包含 `define`, `load` 和 `style` 三个部分.

```Counter.lts
define {
  const count = status<number>(0)
  const increment = () => count.value++
}

load {
  div {
    button.class("button").@click(increment) { "Count is:" count.value }
  }
}

style {
  .button {
    font-weight: bold;
  }
}
```
