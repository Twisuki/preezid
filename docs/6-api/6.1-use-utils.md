# `use` 工具函数

`Preezid` 提供了一系列工具函数, 它们均以 `use` 为前缀, 用于在组件或页面中方便地访问和操作各种功能和数据.

`use` 工具函数有些在编译阶段完成, 有些在运行时完成, 但这里不再需要开发者区分.

## `useBind()`

`useBind()` 是 `Preezid` 提供的用于创建数据绑定的工具函数, 它允许父子组件之间共享和同步数据.

它将返回一个继承 `useStore` 的 `useBind` 对象, 该对象中包含组件中所有需要绑定的数据字段.

它拓展了以下方法:

- `.default(obj: Partial<T>): useBind<T>` - 设置绑定数据的默认值, 仅在父组件未传入该字段时生效.

## `useComponent()`

`useComponent()` 是 `Preezid` 提供的用于声明和注册组件的工具函数.

它将返回一个 `useComponent` 对象, 并使用链式调用的方式直接进行配置.

- `.register(name?: string): void` - 注册组件, 并设置组件名称.
- `.local(path: string): void` - 声明组件为局部组件, 并指定应用路径.
- `.dependent(tyles: DependentNodeType): void` - 声明组件为依赖组件, 并指定依赖类型.

## `useLabel()`

`useLabel()` 是 `Preezid` 提供的用于引入和使用外部组件或页面的工具函数.

它有如下两种用法:

- `useLabel(path: string)` - 组件或页面的相对路径或绝对路径.
- `useLabel(name: string)` - 已注册组件的名称

## `useLocal()`

`useLocal()` 是 `Preezid` 提供的用于创建持久化存储的工具函数. 它会将数据存储在浏览器的 `localStorage` 中, 以便在页面刷新或重新访问时仍能保留数据.

### 参数

- `useLocal(key: string)` - 用于在 `localStorage` 中存储数据的键名.

### 返回值

它将返回一个 `useLocal` 对象, 该对象继承于 `useStore` 对象, 并拓展以下方法:

- `.default(value: T): useLocal<T>` - 设置默认值, 仅在 `localStorage` 中不存在该键时生效.
- `.init(value: T): useLocal<T>` - 初始化值, 无论 `localStorage` 中是否存在该键, 都会将其设置为该值.
- `.remove(): void` - 删除该键及其对应的值.

## `useProps()`

`useProps()` 是 `Preezid` 提供的用于访问组件或页面传入的属性的工具函数.

它将返回一个只读的 `useProps` 对象, 该对象包含组件所有的传入属性, 以及如下方法:

- `.default(obj: Partial<T>): useProps<T>` - 设置属性的默认值, 仅在属性未传入时生效.

## `useStatus()`

_注: `Preezid` 推荐直接简写为 `status()`_

`useStatus()` 是 `Preezid` 提供的用于创建响应式状态的工具函数.

### 参数

- `useStatus(value: T)` - 初始值.

### 返回值

它将返回一个 `useStatus` 对象, 该对象包含以下属性:

- `.value: T` - 当前状态的值.

该对象包含以下方法:

- `.update((prev: T, next: T) => void): void` - 在值更新时调用的回调函数, 接收更新前后的值作为参数.

## `useStore()`

_注: `Preezid` 推荐直接简写为 `store()`_

`useStore()` 是 `Preezid` 提供的用于严格管理状态的工具函数, 适合用于复杂的状态管理场景, 如全局状态管理, 跨组件状态共享, 组件数据绑定等.

### 参数

- `useStore(value: T)` - 初始值.

### 返回值

它将返回一个 `useStore` 对象, 该对象包含以下属性:

- `.value: T` - 当前存储的值, 但为了保证数据安全性, 不可直接访问或修改.

该对象包含以下方法:

- `.get(): T` - 获取当前存储的值.
- `.set(value: T | (self: T) => T): void` - 更新存储的值, 可以直接传入新值, 也可以传入一个函数, 该函数接收当前值作为参数并返回新值.
- `.update((prev: T, next: T) => void): void` - 在值更新时调用的回调函数, 接收更新前后的值作为参数.

该对象包含以下属性:

- `.value: T` - 当前存储的值, 但为了保证数据安全性, 不可直接访问或修改.
