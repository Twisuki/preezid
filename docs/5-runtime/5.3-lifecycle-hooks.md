# 生命周期钩子

由于 `Preezid` 无虚拟 DOM 的概念, 因此组件生命周期十分简单, 只有 `onMount` 和 `onDestroy` 两个钩子.

## `onMount` 挂载钩子

```lts
define {
  onMount(() => {
    console.log("组件已挂载.")
  })
}
```

## `onDestroy` 销毁钩子

```lts
define {
  onDestroy(() => {
    console.log("组件已销毁.")
  })
}
```

## `lifeTimer` 生命周期计时器

我们经常在组件中 `onMount` 时挂载计时器, 在 `onDestroy` 时销毁计时器. 为了简化这一过程, `Preezid` 提供了 `lifeTimer` 函数, 它会创建一个在组件销毁时自动清除的计时器.

```lts
define {
  // 贯穿整个函数周期的计时器
  const timer1 = lifeTimer(() => {
    console.log("每秒输出一次")
  }, 1000)
  
  // 在某个时刻创建的计时器
  let timer2 = null
  const setTimer = () => {
    timer2 = lifeTimer(() => {
      console.log("每两秒输出一次")
    }, 2000)
  }
}
```

它们在组件销毁时都将自动卸载.
